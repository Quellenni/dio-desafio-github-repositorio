O QUE É UM SQL SERVER?  ( Data Base Management System)

- Softwares que padronizam os bancos de dados.
Ex: Arquivo de texto, Excel
- Lidar com padrões de dados armazenados;
- Ser uma interface entre esses dados e sistemas;
- Ser interface entre esses dados e pessoas
- Colocar usuários com acessos e níveis de acesso (excluir dados / inserir dados)

Tipos de Bancos de dados

- SQL SERVER --- =                          |    Sistemas que precisam      |
- ORACLE -------   =    Relacionais     |    de uma confiabilidade        |  
- MYSQL -------- =                             |    muito grande dos dados      |

Atomicidade
- Garante que todos os dados estão corretos, para coloca-los dentro de uma tabela.
